香港親子四天三夜

帶2歲女兒去香港迪士尼

女兒滿兩歲了，這次剛好慶祝老公生日，所以帶老公小孩來香港迪士尼，搭的是從台中直飛的香港快運。 這次四天三夜總結費用時發現這是盤子玩法，分享不害怕當盤子的爸媽們。(花好花滿，只要我還能花，就一定花光光) Day1 迪士尼樂園酒店 check-in 買 2 日票 迪士尼樂園吃晚餐－Corner Cafe，以樂園裡的餐點價格來說，一個有前菜+主餐的 set 約台幣 1200，還在我的預期範圍內。 在看煙火前撤退 Day2 迪士尼樂園吃早餐 (園內的星巴克) 2pm 離場去新酒店-香港維港凱悅尚萃酒店 check in 海港城看聖誕樹 Day3 酒店吃早餐 去中環買蛋塔 晚上去中環看聖誕樹 Day4 回台灣 總花費 機票(2 大 1 小) NTD 29637 迪士尼 2 日票(2 大) NTD 6079 住宿 迪士尼樂園酒店一晚 NTD 19722 維港兩晚 HK 2059.13+2139.88+10%服務費+3%稅 (約 NT 19234) 吃+行 NTD 3 萬 被這趟香港行的消費給驚呆了，沒有預料會花這麼多
